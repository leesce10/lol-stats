# 챔피언 공략 데이터 소스 조사 및 활용 방안

## 1. 현재 한국 공략 생태계 (2025-2026)

### 활성도 순위
```
1위: 유튜브 (도파, 룩삼, 갱맘, 종지호프 등)
2위: OP.GG (통계 기반 메타)
3위: 인벤 (텍스트 공략, 오래된 자료 축적)
4위: 디시 챔피언 갤러리 (실시간 잡담 + 가끔 고퀄)
5위: 나무위키 (메커니즘, 콤보)
6위: 에펨코리아, 클리앙 등 (잡담 위주, 공략 적음)
```

### 핵심 발견
- **유튜브로 대부분 이전됨** — 텍스트 공략 시대는 거의 끝남
- **인벤 챔피언 게시판은 여전히 존재**하지만 활성도는 예전의 30% 수준
- **나무위키**는 패치별로 꾸준히 업데이트됨 (CC 라이선스)
- **챔피언별 디시 마이너 갤러리** (예: 야스오 갤)이 의외로 깊은 정보 제공

## 2. 데이터 소스별 활용 가능성

### A. 유튜브 (가장 현실적, 합법적) ⭐⭐⭐⭐⭐

**YouTube Data API v3**
- 무료, 일 10,000 요청
- 합법적
- 가져올 수 있는 데이터:
  - 영상 제목, 설명, 태그
  - 조회수, 좋아요, 댓글 수
  - 채널 정보
  - 썸네일
  - 자막 목록 (내용은 제한적)
- 가져올 수 없는 것:
  - 영상 원본 다운로드 (ToS 위반)

**활용 방법**
```
"야스오 미드 공략" 검색 → 영상 5개 자동 추출
→ 우리 사이트의 야스오 페이지에 임베드
→ 유저가 클릭하면 유튜브로 이동
```

**비용**: 0원
**합법성**: 100% 합법 (검색 + 임베드는 YouTube가 권장)

### B. 나무위키 ⭐⭐⭐⭐

**라이선스**: CC BY-NC-SA 2.0 KR
- **비상업적 이용 가능**
- **출처 표기 필수**
- **상업적 사용 시 별도 협의 필요**

**활용 방법**
1. 챔피언별 페이지에서 **콤보, 스킬 메커니즘, 운영 팁** 추출
2. 우리 사이트에 인용 표시 + 나무위키 링크
3. 유저가 더 자세한 정보 원하면 → 나무위키로 이동

**기술적 구현**
- 나무위키 자체는 API 없음
- HTML 파싱 필요
- 라이선스 조건 준수하면 합법

**현재 상태에서**: 개인 테스트 단계니 일단 가능. 상업화 시점에 협의 필요.

### C. 인벤 ⭐⭐⭐

**상황**
- 챔피언별 공략 게시판: https://lol.inven.co.kr/dataninfo/champion/
- 활성도 떨어졌지만 **오래된 고퀄 가이드 축적**
- 특정 챔피언 베스트글은 여전히 가치 있음

**합법성**
- 게시글 자체는 작성자 저작권
- **링크 + 제목만 가져오는 건 OK**
- 본문 가공은 위험

**현실적 활용**
- 챔피언 페이지에서 "인벤 추천 공략" 섹션
- 제목 + 작성자 + 추천수만 보여주고 → 클릭 시 인벤으로 이동
- 아무 본문도 가져오지 않음

### D. Riot Data Dragon ⭐⭐⭐⭐⭐ (이미 사용 중)

- 공식 API, 100% 합법, 무료
- 챔피언 기본 정보 (스킬 설명, 스탯)
- 우리가 이미 사용 중

### E. 디시 챔피언 갤러리 ⭐⭐

- 마이너 갤러리 형태로 챔피언별 존재
- 예: 야스오 갤, 제드 갤
- 가끔 진짜 고퀄 가이드 글이 올라옴
- 합법적 수집 어려움

## 4. 구현 우선순위

### Phase 1 (가장 쉽고 효과 큼) — YouTube 임베드
- YouTube Data API 키 발급 (Google Cloud, 무료)
- 챔피언별 검색어 자동 생성: "{챔피언명} 공략", "{챔피언명} 가이드"
- 인기 영상 5개 추출 → 우리 페이지에 임베드
- **개발 시간**: 2~3일
- **비용**: 0원

### Phase 2 — 나무위키 핵심 정보 추출
- 나무위키 챔피언 페이지 파싱 (HTML)
- "콤보", "운영", "팁" 섹션 자동 추출
- 출처 표기 후 우리 사이트에 표시
- **개발 시간**: 1주
- **법적 주의**: CC BY-NC-SA, 상업화 전 검토

### Phase 3 — 인벤 추천 공략 링크
- 인벤 챔피언 게시판 베스트글 스크래핑
- 제목 + 작성자 + 추천수만 표시
- 본문은 가져오지 않음, 클릭 시 인벤으로 이동
- **개발 시간**: 1주
- **법적 주의**: robots.txt 준수, 본문 미수집

### Phase 4 — LLM 핵심 요약
- 챔피언별 1줄 요약 + 상세 운영법
- Claude Haiku로 생성, 캐시
- **개발 시간**: 3일
- **비용**: 약 $1 (168개)

## 5. 법적 주의사항 (중요)

### 지금 (개인 테스트 단계)
- **거의 모든 방법 시도 가능**
- robots.txt만 안 어기면 OK
- 외부 공개 안 하는 한 문제 없음

### 공개 홍보 전에 반드시
1. 모든 외부 데이터 출처 표기
2. 본문 직접 게재 금지 (제목 + 링크만)
3. 나무위키 비상업적 이용 조건 확인
4. 인벤/디시 robots.txt 확인
5. YouTube ToS 준수 (영상 다운로드 절대 금지)

### 상업화 시점
- 인벤, 나무위키와 공식 협의 필요
- 또는 모든 외부 데이터 제거하고 자체 콘텐츠로 대체
- YouTube, Riot API는 그대로 사용 가능 (공식 허용)

## 6. 결론 및 추천

### 가능한가? **예. 충분히 가능합니다.**

가장 현실적인 방법:
1. **YouTube API**로 챔피언별 인기 영상 자동 추천 (즉시)
2. **LLM**으로 챔피언 핵심 요약 + 콤보 정리 (자체 콘텐츠)
3. **나무위키** 보조 정보 인용 (출처 표기)
4. **인벤** 글 링크만 (본문 X)

### 추천 접근

**"데이터를 직접 가공해서 우리 콘텐츠로 만든다"**보다는

**"각 사이트의 정보를 한 곳에 모아서 보여준다"** 가 법적/현실적으로 안전합니다.

```
우리 사이트 = "롤 정보 허브"
- 통계: op.gg/lol.ps
- 영상: YouTube
- 텍스트 공략: 인벤 링크
- 메커니즘: 나무위키 인용
- 자체 가이드: LLM 생성
```

이렇게 하면:
- 합법적
- 유저는 한 곳에서 모든 정보 접근
- 우리는 거대 사이트의 일부 콘텐츠 차용 안 함 (링크만)
- 차별화: "다른 사이트는 한 가지만, 우리는 모두"
