# 챔피언 가이드 제작 과정 문서

> 10개 챔피언(제드, 바이, 미스 포츈, 이즈리얼, 제리, 드레이븐, 징크스, 애쉬, 아리, 제이스)의 가이드가 어떤 과정으로 만들어졌는지 추적 기록.

---

## 전체 파이프라인

```
1. YouTube 검색 (자동)
   └─ scripts/search-youtube.ts
2. 자막 추출 (자동)
   └─ youtube-transcript 라이브러리
3. 자막 데이터 저장
   └─ scripts/data/{Champion}-raw.json
4. 가공 (수동 + 메타 지식 종합)
   └─ src/data/champion-guides.ts
5. 챔피언 상세 페이지 표시
   └─ src/app/stats/[champion]/page.tsx
```

---

## 공통 수집 과정

### 1단계: 검색어로 영상 검색
- 챔피언당 2~4개의 검색어 사용
- 예: "제드 미드 강의", "제드 라인전 강의", "제드 운영 강의", "롤 제드 강의"
- 각 검색어마다 상위 15~20개 영상 후보 추출

### 2단계: 영상 필터링
- **재생 시간**: 5~30분 (쇼츠/광고 제외)
- **조회수**: 10,000회 이상
- **제목 필터**: "오버워치", "와일드리프트" 제외
- **점수 가중치**: "강의" 1.5배, "공략" 1.3배, "장인" 1.2배, "기초/입문" 1.2배

### 3단계: 자막 추출
- youtube-transcript 라이브러리로 한국어 자동 자막 추출
- 자막 없는 영상은 자동 제외

### 4단계: 가공
- 여러 영상의 자막을 종합하여 메타 지식 기반으로 정리
- 출처 영상 모두 표기 (제목, 채널, 영상 ID)

- 멀티 모드 활용 4개 콤보 작성

## 가공 방법론

### 자막 → 가이드 변환 원칙

**1. 직접 인용 금지**
- 자막을 그대로 복사하지 않음
- 핵심 메시지만 추출해서 재작성

**2. 다중 출처 합성**
- 여러 영상의 공통된 핵심을 우선
- 한 영상에만 나오는 정보는 검증 후 채택
- 모순되는 정보는 다수 의견 채택

**3. 메타 지식 결합**
- 자막에 없는 일반적 챔피언 정보(룬/아이템)는 메타 지식으로 보완
- 자막에서 추출한 부분과 메타 지식 부분 모두 가이드의 일부

**4. 출처 표기 필수**
- 모든 가이드의 sources 배열에 영상 ID, 제목, 채널, 조회수 명시
- 챔피언 페이지에서 영상 썸네일 + 직접 링크 제공
- 트래픽을 원본 영상으로 환원

### 가공된 데이터 구조

각 챔피언 가이드는 다음 필드를 포함:

```typescript
{
  championId, championKr, position,
  oneLine,             // 한 줄 핵심
  difficulty,          // 1~5 난이도
  damageType,          // AD/AP/Mixed
  strengths[],         // 강점 3~4개
  weaknesses[],        // 약점 3~4개
  combos[],            // 핵심 콤보 2~4개
  earlyGame[],         // 라인전 운영 5개
  midGame[],           // 중반 운영 5개
  lateGame[],          // 후반 운영 5개
  tips[],              // 실전 팁 5개
  goodAgainst[],       // 강한 상대 5개
  badAgainst[],        // 약한 상대 5개
  recommendedRunes,    // 추천 룬
  recommendedItems,    // 추천 아이템
  sources[],           // 출처 영상 (필수)
}
```

## 한계점 및 향후 개선

### 현재 한계
1. **자막 자동 인식 오류**: 일부 영상은 자막 품질이 매우 낮아 활용 불가
2. **수동 가공**: 자동화되지 않아 시간 소요
3. **메타 변화 미반영**: 패치 후 자동 갱신 필요
4. **저작권 회색지대**: 상업화 시 영상 저작권자 사전 협의 필요

### 개선 방향
1. **LLM 자동 가공**: Claude/GPT API로 자막 → 가이드 자동 변환
2. **패치 자동 갱신**: 새 패치 출시 시 영상 재수집 + 가이드 업데이트
3. **유튜버 협업**: 인기 채널과 공식 협의 (트래픽 환원 모델)
4. **유저 피드백**: "이 가이드 도움됐어요" 평가 수집
