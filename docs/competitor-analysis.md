# 경쟁사 분석

---

## 1. OP.GG — 시장 지배자

### 핵심 기능
- 소환사 전적 검색 (KDA, CS, 포지션별 통계)
- 챔피언 티어/승률/픽률/밴률
- 라이브 게임 팀원/상대 즉시 조회
- 멀티서치 (여러 소환사 동시 검색)
- OP Score (자체 퍼포먼스 점수)

### 차별화 포인트
- 브랜드 파워: "롤 전적 = OP.GG" 인식
- 글로벌 + 멀티게임 (발로란트, TFT, 오버워치)
- 인게임 오버레이 앱, 크롬 확장프로그램

### 수익 모델
- 배너 광고 (메인)
- OP.GG Premium (광고 제거, 심화 통계)
- 라이엇 공식 파트너십

### 유저가 가장 많이 쓰는 기능
라이브 게임 조회 > 소환사 전적 > 챔피언 티어

---

## 2. U.GG — 빌드 특화

### 핵심 기능
- 챔피언 최적 빌드 (승률 기반 룬/아이템 추천)
- ELO 구간별 세분화 티어 리스트 (챌린저~아이언)
- 포지션 매치업 (카운터 픽 추천)
- 프로/챌린저 빌드 트래킹

### 차별화 포인트
- 데이터 신뢰성 강조 ("수백만 게임 기반", 샘플 수 명시)
- ELO 구간 필터링 → 고랭크 유저에게 인기
- 해외(NA/EUW) 점유율 높음

### 수익 모델
- 광고 + U.GG Pro (광고 제거, 상세 필터)

### 유저가 가장 많이 쓰는 기능
챔피언 빌드 > ELO별 티어리스트 > 카운터 픽

---

## 3. LeaguePHD — 심층 분석

### 핵심 기능
- 스킬 사용 패턴, 아이템 구매 순서 통계
- 분당 골드/CS/레벨 성장 곡선 시각화
- 선픽 vs 후픽, 솔로 vs 파티 분리 통계
- 패치별 승률 변화 추이 그래프

### 차별화 포인트
- "박사(PhD)" 컨셉 — 데이터를 더 깊이 파는 분석
- 타임라인 분석 (초반/중반/후반 구간별 성능)
- 데이터 덕후 유저층 확보

### 수익 모델
- 광고 기반 (프리미엄 미비)

### 유저가 가장 많이 쓰는 기능
심층 챔피언 분석 > 패치별 통계 변화

---

## 4. Lolalytics — 세분화 데이터

### 핵심 기능
- 스킬 맥스 순서별 승률, 아이템 조합별 승률 초세분화
- 정글 캠프 순서/갱킹 경로 통계
- 시너지/카운터 매트릭스

### 차별화 포인트
- 가장 세분화된 데이터 (다른 사이트가 안 보여주는 미세 변수)
- 정글 특화 (캠프 순서/효율)
- 패치 직후 반영 빠름

### 수익 모델
- 광고 + 프리미엄 (심화 필터링)

### 유저가 가장 많이 쓰는 기능
챔피언 세부 스킬 통계 > 정글 패스 > 아이템 세부 데이터

---

## 5. Porofessor.gg — 라이브 게임 특화

### 핵심 기능
- 챔피언 선택 화면에서 즉시 팀원 분석
- 인게임 오버레이 앱 (자동으로 팀원 정보 표시)
- 팀원 전체 실력 점수/등급 요약

### 차별화 포인트
- 라이브 게임 분석에 올인
- 앱 경험 최적화 (별도 검색 불필요)
- "적을 알고 나를 알자" 컨셉

### 수익 모델
- 광고 + Porofessor Premium

### 유저가 가장 많이 쓰는 기능
라이브 게임 팀원 분석 > 인게임 오버레이

---

## 6. FOW.KR (포우) — 국내 특화

### 핵심 기능
- 한국 서버 전적 검색
- 프로게이머 솔로랭크 계정 모니터링
- 챌린저/그랜드마스터 랭킹

### 차별화 포인트
- 국내 커뮤니티(디시, 클리앙) 연계성
- 프로게이머 추적
- 심플하고 빠른 UI

### 수익 모델
- 광고 기반

---

## 공통 성공 패턴

| 패턴 | 설명 |
|------|------|
| 라이브 게임 연동 | 게임 직전 팀원/상대 파악 → 가장 높은 사용 동기 |
| 빠른 패치 반영 | 새 패치 후 수 시간 내 업데이트 → 재방문 유도 |
| 무료 기본 + 유료 심화 | 핵심은 무료, 광고 제거/심화 데이터는 유료 |
| 데이터 시각화 | 숫자 나열이 아닌 그래프, 색상, 아이콘으로 직관성 |
| 니치 공략 | 1위가 있어도 특정 기능 특화로 생존 |
| 커뮤니티 바이럴 | 링크가 자연스럽게 공유되는 구조 |

## 유저 공통 불만

### 데이터
- 비주류 챔피언 통계 왜곡 (샘플 부족)
- 패치 직후 통계 신뢰도 낮음
- 서버별 메타 차이 미반영

### UX
- 광고 과다 → 정보 접근 방해
- 광고 스크립트로 로딩 지연
- 모바일에서 정보 잘림

### 기능
- 프로필 수동 업데이트 (자동 갱신 없음)
- 라이브 게임 서버 과부하 시 로딩 실패

---

## 핵심 인사이트

> 이 시장은 "데이터 자체"보다 **"언제, 어떤 맥락에서 보여주느냐"**가 차별화의 핵심.

3가지 사용 맥락:
1. **게임 직전** — 라이브 게임, 상대 분석 (Porofessor가 강함)
2. **게임 준비 중** — 빌드, 티어, 카운터 (U.GG가 강함)
3. **게임 후 복기** — 전적, 통계 (OP.GG가 강함)

## 우리 서비스 포지셔닝

```
                  데이터 깊이
                     ↑
         Lolalytics  │  LeaguePHD
                     │
    ─────────────────┼─────────────────→ 사용 편의성
                     │
          FOW.KR     │  ★ LOL Stats (목표 위치)
                     │
         OP.GG       │  Porofessor
                     │
```

**목표**: 사용 편의성은 높으면서, 맞라인 가이드/조합 분석이라는 특화 영역에서 데이터 깊이를 확보
