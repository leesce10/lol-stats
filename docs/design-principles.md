# 서비스 기획/디자인 원칙

## 기획 철학
> "챔피언 셀렉트 70초 안에 답을 줄 수 있는 서비스"

모든 기능은 이 질문을 통과해야 한다:
1. **유저가 언제 이걸 쓰는가?** (게임 전/중/후)
2. **10초 안에 원하는 답을 찾을 수 있는가?**
3. **이 기능이 실제로 게임에 도움이 되는가?**

---

## UX 원칙

### 1. 속도가 기능이다
- 페이지 로딩 2초 이내
- 검색 결과 즉시 표시
- 불필요한 애니메이션 제거

### 2. 정보 계층 구조
- **1단계**: 핵심 숫자 (승률, 티어) — 0.5초에 파악
- **2단계**: 비교 데이터 (매치업, 조합) — 3초에 파악
- **3단계**: 상세 분석 (가이드, 팁) — 관심 있으면 읽기

### 3. 모바일 우선
- 테이블은 가로 스크롤 가능하게
- 터치 타겟 최소 44px
- 챔피언 선택은 그리드 형태 (아이콘 터치)

### 4. 데이터 신뢰도 표시
- 항상 샘플 수 표시 ("1,234경기 기준")
- 데이터 최종 업데이트 시간 표시
- 샘플 부족 시 명확히 경고

### 5. 광고와 공존
- 핵심 정보 영역에는 광고 배치 안 함
- 사이드바, 페이지 하단, 페이지 전환 시 자연스럽게
- 광고 때문에 레이아웃 밀리지 않게 고정 영역 확보

## 기능 우선순위 매트릭스

| | 구현 쉬움 | 구현 어려움 |
|------|---------|---------|
| **임팩트 높음** | 즉시 구현 | 계획 후 구현 |
| **임팩트 낮음** | 여유 있을 때 | 안 함 |

### 즉시 구현 (쉬움 + 높은 임팩트)
- 챔피언 티어 리스트
- 소환사 검색
- 챔피언 빌드 추천

### 계획 후 구현 (어려움 + 높은 임팩트)
- 실시간 인게임 분석
- AI 기반 승률 예측
- 데이터 자동 수집 파이프라인

### 여유 있을 때 (쉬움 + 낮은 임팩트)
- 무료 챔피언 로테이션 표시
- 다크/라이트 테마 전환
- 다국어 지원

### 안 함 (어려움 + 낮은 임팩트)
- 리플레이 분석
- 커스텀 게임 통계
- 프로 경기 분석

## 경쟁사 대비 차별화 전략

### 우리만의 강점으로 밀 것
1. **맞라인 실전 가이드** — 단순 승률이 아닌 "어떻게 플레이하라"까지
2. **조합 분석기** — 픽 완료 후 바로 조합 평가
3. **심플한 UI** — op.gg의 정보 과잉 문제 해결

### 경쟁사와 동일하게 갖출 것 (기본기)
- 소환사 전적 검색
- 챔피언 티어 리스트
- 빌드/룬 추천

### 경쟁사와 싸우지 않을 것
- 프로 경기 분석 (op.gg가 압도적)
- 커뮤니티/소셜 기능 (별도 플랫폼)
