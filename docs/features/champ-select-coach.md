# 챔셀 코치 (Champion Select Coach) 기획서

> 솔랭 챔셀 70초 안에 이 판을 어떻게 이길지 알려주는 도구. 우리 사이트의 핵심 차별화.

---

## 1. 한 줄 정의

> **"이 10명 픽 조합에서 내가 어떻게 플레이해야 이기는가"** 를 답하는 도구.

---

## 2. 사용 시나리오

### 주요 사용자
- **솔랭 트라이하더** (1순위) — 매 게임 이기고 싶은 유저
- **티어 올리고 싶은 모든 유저**

### 사용 시점
- **솔랭 챔셀** — 모든 픽이 끝나가는 60초 시점
- 게임 시작 직전 마지막 10초 확인

### 사용 시간
- **목표: 30초 안에 핵심 가이드 전달**
- 입력: 5~15초 (입력 방식에 따라)
- 결과 인지: 15~25초 (핵심만 빠르게 스캔)

### 진짜 사용 흐름
```
챔셀 시작 (0초)
↓
포지션 정해짐 (10초)
↓
양팀 픽 진행 (10~50초)
↓
60초 시점 → 우리 사이트로 이동
↓
10명 빠르게 입력 (60~75초)
↓
한 줄 핵심 확인 + 시간대별 가이드 스캔 (75~90초)
↓
챔셀 종료 (게임 시작)
```

## 3. 입력 방식 (3가지 동시 지원)

### A. 키보드 빠른 입력 (기본) ⭐⭐⭐
- 슬롯 10개 (블루팀 5 + 레드팀 5)
- 각 슬롯에 1글자만 입력 → 자동완성 → Enter → 다음 슬롯
- **목표: 15초 안에 10명 입력**

### B. 캡처 + 붙여넣기 (Phase 2) ⭐⭐⭐⭐⭐
- 챔셀 화면 캡처 (Win+Shift+S)
- 우리 사이트에서 Ctrl+V
- AI Vision API가 10명 자동 인식
- **목표: 3초 안에 끝**
- 비용: $0.001/분석 (OpenAI Vision)

### C. 소환사명 자동 인식 (Phase 3, 궁극 목표) ⭐⭐⭐⭐⭐
- 자기 소환사명 1번 등록
- 챔셀 들어가면 자동 감지 (Riot Spectator API)
- **목표: 0초 입력**
- **선결 조건: Riot Production API Key**

## 4. 출력 영역 — 단계별 정보

### 화면 구성 (위에서 아래로)

#### 1️⃣ 즉시 표시 (1초 안에 봐야 할 정보)

**한 줄 핵심**
```
💡 15분 안에 끝내야 합니다. 적이 풀템 가면 못 이깁니다.
```

**우리팀 vs 적팀 전략**
```
우리: 다이브/한타팀 (자르반, 사이온)
적군: 후반 스케일링팀 (베인, 케일)
→ 빠른 전개 필수
```

#### 2️⃣ 매치업 (내 라인 한정)

**내 라인 매치업**
```
🆚 미드 매치업
제드 51.3% vs 야스오 48.7%
💡 야스오 점화 쿨타임 추적이 핵심
```

#### 3️⃣ 시간대별 행동 지침

**🌅 라인전 (1~14분)**
```
✓ 우리 정글 자르반과 6분, 9분 갱크 콜
✓ 6레벨 솔킬보다 협동 다이브 우선
✓ 솔킬 노리지 마세요. 점화로 마무리만
```

**⚔️ 중반 (14~25분)**
```
✓ 모든 한타에 합류 (스플릿 금지!)
✓ 적 베인 풀템 전 한타 강제
✓ 드래곤보다 적 미드/탑 컷 우선
```

**🏆 후반 (25분+)**
```
✓ 한타 회피, 슈퍼미니언 푸시
✓ 베인 컷 우선 (우리 탱이 사이온 1명뿐)
✓ 미스나면 즉시 백
```

#### 4️⃣ 흔한 함정 3가지

```
⚠️ 함정 1: "내가 카운터니까 솔킬 가능"
→ 야스오는 풀템에 더 강함. 라인전만 이겨도 후반 약함

⚠️ 함정 2: 6레벨 즉시 궁 사용
→ 야스오 W로 차단. 9레벨 이후 사용 권장

⚠️ 함정 3: 우리 정글 안 부르고 솔킬 노림
→ 6~10분이 골든타임. 13분 후엔 늦음
```

#### 5️⃣ 빌드 카운터 추천

```
🛒 이 매치업 + 적팀 빌드 카운터
1. 시야: 핑크와드 + 충격포자 (5분~)
2. 1코어: 크라켄 슬레이어
3. 2코어: 가시갑옷 (베인 견제)
4. 3코어: 죽음의 무도

❌ 피해야 할: 마법저항 (적팀 다 AD)
```

#### 6️⃣ 한타 우선순위

```
🎯 한타 시 컷 우선순위
1. 베인 (적 메인 캐리, 후반 핵심)
2. 카르마 (적 서폿, 베인 보호)
3. 야스오 (적 미드, 풀템 후 위협)

⚠️ 절대 진입 금지
- 적 사이드에 룰루 + 베인 둘 다 있을 때
```

## 6. 분석 로직

### 6-1. 우리팀 / 적팀 성향 분석

**챔피언 속성 (사전에 168개 챔프에 1번 부여)**
```typescript
type ChampionTraits = {
  damage: 'AD' | 'AP' | 'mixed';
  role: 'tank' | 'fighter' | 'mage' | 'assassin' | 'marksman' | 'enchanter';
  earlyGame: 'strong' | 'medium' | 'weak';
  lateGame: 'strong' | 'medium' | 'weak';
  engage: 'strong' | 'medium' | 'none';
  splitpush: 'strong' | 'medium' | 'none';
  poke: 'strong' | 'medium' | 'none';
  protection: 'strong' | 'medium' | 'none';
  range: 'melee' | 'ranged';
  cc: 'high' | 'medium' | 'low';
};
```

**팀 성향 자동 도출**
```typescript
function analyzeTeamComp(picks: ChampionTraits[]) {
  return {
    composition: 
      hasEngage(picks) && hasFrontline(picks) ? 'engage_team' :
      hasPokers(picks) && hasRanged(picks) ? 'poke_team' :
      hasSplitter(picks) ? 'splitpush_team' :
      'standard',
    powerSpike: 
      countLateGame(picks) >= 3 ? 'late' :
      countEarlyGame(picks) >= 3 ? 'early' :
      'mid',
    frontline: countTanks(picks) + countTankFighters(picks),
    damageBalance: getDamageBalance(picks),
  };
}
```

**전략 도출**
```typescript
function getStrategy(myTeam, enemyTeam) {
  if (myTeam.powerSpike === 'late' && enemyTeam.powerSpike === 'early') {
    return "버티고 후반 가세요";
  }
  if (myTeam.powerSpike === 'early' && enemyTeam.powerSpike === 'late') {
    return "15분 안에 끝내야 합니다";
  }
  if (myTeam.composition === 'engage_team' && enemyTeam.composition === 'poke_team') {
    return "한타 강제, 포크 회피";
  }
  // ...
}
```

## 9. 차별화 포인트

### vs 경쟁사 모두
| 사이트 | 챔셀 분석 |
|--------|----------|
| op.gg | 멀티서치 (소환사 정보만) |
| lol.ps | 없음 |
| Porofessor | 라이브 게임 자동 분석 (영어, 한국어 미흡) |
| **우리** | **10명 조합 + 매치업 + 시간대별 행동 (한국어, 모바일 최적화)** |

### 우리만의 강점
1. **시간대별 행동 지침** — 누구도 안 함
2. **흔한 함정 경고** — 일반 통계로 못 줌
3. **빌드 카운터 추천** — 매치업+조합 둘 다 반영
4. **모바일 최적화** — 휴대폰 들고 챔셀 가능
5. **광고 없는 빠른 로딩**
