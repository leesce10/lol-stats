# 맞라인 비교 (Matchup Comparison) 기획서

> 가벼운 매치업 정보 도구. 5초 안에 답을 준다.

---

## 1. 한 줄 정의

> **"이 챔프 vs 저 챔프, 어떻게 해야 이기지?"** 에 5초 안에 답하는 도구.

---

## 2. 사용 시나리오

### 주요 사용자
- **솔랭 트라이하더 (평상시)** — 챔프 공부할 때
- **입문/복귀 유저** — 자기 챔프가 누구한테 강한지 알고 싶을 때
- **데이터 덕후** — 매치업 통계 확인용

### 사용 시점
- 게임 끝나고 복기할 때
- 새로운 챔프 배울 때
- 친구랑 노말/롤체 가는 길에 가볍게
- 챔셀 시간이 부족할 때 (10명 입력은 부담)

### 사용 시간
- **목표: 5초 안에 핵심 정보 전달**
- 입력: 2초 (챔프 2개 선택)
- 결과 인지: 3초 (한 줄 핵심 + 통계)

---

## 3. 핵심 기능

### 입력 영역
- **포지션 선택** (탑/정글/미드/바텀) — 4개 (서폿은 바텀에 통합)
- **내 챔피언** 선택
- **상대 챔피언** 선택
- 빠른 입력: 키보드 1글자 자동완성

### 출력 영역 — 4가지 핵심만

#### A. 한 줄 핵심 ⭐⭐⭐ (가장 중요)
이 매치업의 본질을 1문장으로.
```
💡 야스오 점화 쿨타임 추적이 승패를 결정합니다.
```

#### B. 매치업 승률 (작게)
```
제드 51.3%  vs  야스오 48.7%
(8,234 경기 기준 · op.gg KR Emerald+)
```

#### C. 카운터/이지매치업 (양쪽 5개씩)
- 내 챔프의 카운터 5개
- 내 챔프가 강한 상대 5개
- 클릭하면 그 매치업으로 즉시 이동

#### D. 플레이 팁 (3개)
- 라인전 핵심 1개
- 6레벨 후 핵심 1개
- 후반/운영 핵심 1개

## 5. 입력 UX 최적화

### 키보드 빠른 입력
```
순서:
1. 페이지 진입 → 자동으로 "내 챔피언" 검색창 포커스
2. "ㅈ" 입력 → "제드" 강조
3. Enter → 다음 슬롯 자동 이동
4. "야" 입력 → "야스오" 강조
5. Enter → 즉시 결과 표시

총: 4번 키 입력 = 2초
```

### 자동완성 동작
- 한글 1글자만으로 매칭
- 첫 글자 일치 + 픽률 높은 순으로 정렬
- Tab으로 다음 후보, Enter로 선택

### 모바일 입력
- 챔피언 그리드 (이미 구현됨)
- 검색창 + 빠른 선택 모두 지원

## 6. 데이터 출처

| 데이터 | 출처 | 갱신 |
|--------|------|------|
| 챔프 기본 통계 | op.gg/lol.ps 스크래핑 | 패치별 |
| 매치업 승률 | op.gg/lol.ps | 패치별 |
| 카운터/이지매치업 | 우리 알고리즘 (이미 구현) | 패치별 |
| 한 줄 핵심 | LLM 생성 + 사람 검수 | 1회 생성, 영구 |
| 플레이 팁 3개 | LLM 생성 + 캐시 | 1회 생성, 영구 |

### 한 줄 핵심 + 팁 생성 방식
- Claude Haiku에 "이 매치업의 핵심 1줄 + 라인전/6렙후/운영 팁 1개씩 생성" 요청
- Supabase `matchup_guides` 테이블에 캐시
- 한 매치업 당 약 $0.001
- 인기 매치업 약 2,000개 사전 생성 → 약 $2

## 7. 데이터 모델

```sql
CREATE TABLE matchup_guides_lite (
  id BIGSERIAL PRIMARY KEY,
  champion_a TEXT NOT NULL,
  champion_b TEXT NOT NULL,
  position TEXT NOT NULL,
  one_line_tip TEXT,        -- 한 줄 핵심
  early_tip TEXT,            -- 라인전 팁
  mid_tip TEXT,              -- 6렙 후 팁
  late_tip TEXT,             -- 운영 팁
  generated_at TIMESTAMPTZ DEFAULT NOW(),
  UNIQUE(champion_a, champion_b, position)
);
```

## 8. 차별화 포인트

### vs 경쟁사
| 사이트 | 매치업 정보 | 단점 |
|--------|-----------|------|
| op.gg | 통계만 | "그래서 어떻게?" 답 없음 |
| lol.ps | 통계 + 카운터 | 텍스트 가이드 없음 |
| 우리 | 통계 + 한 줄 핵심 + 플레이 팁 | ✅ |

### 우리만의 강점
1. **한 줄 핵심**이 사이트의 트레이드마크
2. **5초 안에 답** 보장
3. **카운터 클릭 → 즉시 이동**으로 빠른 탐색
