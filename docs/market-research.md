# 한국 롤 정보 사이트 시장 조사 (2025-2026)

## 시장 인지도 순위

```
1위  OP.GG          — 압도적 지배 (사실상 표준)
2위  DAK.GG         — TFT+LoL 통합으로 급성장 ⬆
3위  DEEPLOL.GG     — AI 분석으로 틈새 확보 ⬆
4위  FOW.KR         — 한국 특화, 정체
5위  LOL.PS         — 소규모 유지
6위  U.GG           — 빌드 참고용 한정 이용
7위  POROFESSOR.GG  — 인게임 특화 사용자
8위  LOLALYTICS     — 마니아층
9위  MOBALYTICS     — 한국 거의 미이용
```

## 주요 사이트 상세

### 1. OP.GG — 시장 지배자
- **URL**: op.gg
- **인지도**: 매우 높음 (한국 일평균 40~60만)
- **수익**: 광고 + Premium 구독
- **핵심**: 소환사 전적 검색의 표준, Riot 공식 파트너
- **강점**: 압도적 브랜드 파워, 모바일 앱 완성도, 글로벌 통합
- **약점**: AI 분석 약함, 광고 많음, 차별화 정체

### 2. DAK.GG — 급성장 (TFT 강세)
- **URL**: dak.gg
- **인지도**: 높음 (10~30만 추정)
- **수익**: 광고 + 프리미엄
- **핵심**: TFT 정보 분야 국내 1위, LoL+TFT 통합 플랫폼
- **강점**: TFT 독보적, 세련된 UI, 빠른 패치 대응
- **약점**: LoL 전적은 OP.GG에 밀림

### 3. DEEPLOL.GG — AI 분석 특화
- **URL**: deeplol.gg
- **인지도**: 중간~높음 (5~15만)
- **수익**: 광고 + 프리미엄
- **핵심**: AI 기반 약점/강점 분석, "왜 졌는가" 데이터 설명
- **강점**: AI 분석이 실용적, 자기개선 도구
- **약점**: 일부 분석 정확도 논란, 학습 곡선

### 4. FOW.KR — 한국 특화 정체
- **URL**: fow.kr
- **인지도**: 중간~높음 (5~15만)
- **핵심**: KR 서버 특화, 국내 스트리머/프로 추적
- **강점**: 무료 기능 범위 넓음, 빠른 업데이트
- **약점**: 모바일 미흡, 차별화 부족, 정체 중

### 5. LOL.PS — 상성 데이터 특화
- **URL**: lol.ps
- **인지도**: 중간 (3~8만)
- **핵심**: 챔피언 상성 데이터 상세, 패치별 변화 추적
- **강점**: 상성 데이터, 빠른 로딩
- **약점**: 기능 다양성 부족, 모바일 앱 없음

### 6. U.GG — 빌드 통계 (해외 강세)
- **URL**: u.gg
- **인지도**: 중간 (한국 2~5만)
- **핵심**: 챌린저 메타 기반 빌드 추천, 랭크별 분리 통계
- **강점**: 빌드 통계 신뢰도, 빠른 로딩
- **약점**: 한국어 미지원, 한국 서버 정보 부족

### 7. POROFESSOR.GG — 인게임 특화
- **URL**: porofessor.gg
- **인지도**: 중간 (한국 2~5만)
- **핵심**: 챔피언 선택 화면 자동 분석
- **강점**: 인게임 실시간 분석
- **약점**: 한국어 지원 미흡

### 8. LOLALYTICS — 마니아용 심층 데이터
- **URL**: lolalytics.com
- **인지도**: 낮음 (한국 1~2만)
- **핵심**: 통계 깊이 압도적, 아이템 조합별 승률
- **강점**: 무료 고급 통계, 데이터 깊이
- **약점**: UI 불친절, 한국어 없음

### 9. MOBALYTICS — 한국에서 쇠퇴
- **URL**: mobalytics.gg
- **인지도**: 낮음 (한국 1만 미만)
- **핵심**: GPI(Gamer Performance Index), 8가지 역량 지표
- **강점**: 코칭 플랫폼 정체성
- **약점**: 한국어 거의 없음, 핵심 기능 유료

## 우리의 차별화 전략 (3가지 핵심)

### 1️⃣ "밴픽 단계"에 올인
**문제**: OP.GG는 게임 후 복기 도구. 밴픽 중에 빠르게 의사결정 도움을 주는 사이트는 없음.

**해결**: 챔피언 선택 화면 70초 안에 모든 판단을 끝낼 수 있는 UX
- 양팀 픽 입력 → 즉시 승률/전략 가이드
- 카운터/시너지 한눈에
- 모바일 우선 (휴대폰 들고 게임)

**경쟁사 대비**: OP.GG는 데스크탑 앱으로만 가능, 우리는 웹/모바일에서 빠르게

### 2️⃣ 한국어 + 모바일 + 광고 없이 빠르게
**문제**: 좋은 사이트는 영어이고, 한국어 사이트는 광고/속도가 느림.

**해결**: 핵심 영역에는 광고 없음, 모바일 최적화, 100% 한국어
- 무료 사용자도 핵심 기능 광고 없이
- 광고는 사이드/하단에만

### 3️⃣ AI를 단순 분석이 아닌 "실용적 도구"로
**문제**: DeepLoL의 AI는 분석은 잘 하지만 너무 복잡함.

**해결**: 단순하고 실용적인 AI 기능
- **AI 챗봇 코치**: 자연어로 "야스오 카운터 누구?" 질문 가능
- **컨디션 트래커**: "오늘 게임 그만하세요" 알림 (3연패 + 새벽 = 위험)
- **메타 예측**: "다음 패치에서 이 챔프 너프 가능성 높음"
- **나만의 약점 1줄 요약**: "당신의 약점: 라인전 CS"

## 우선순위 추천 (현실적 ROI 기반)

### Phase 1 — 차별화 MVP (지금 ~ 1개월)
1. **밴픽 단계 분석** — 양팀 픽 입력 → 즉시 가이드 (이미 있음, 강화)
2. **모바일 완벽 대응** — 광고 없이 빠르게
3. **컨디션 트래커** — 시간대별 승률 분석
4. **AI 챗봇 코치 (간단 버전)** — 자주 묻는 질문 자동 답변

### Phase 2 — 기능 확장 (1~3개월)
1. **소환사 검색 기본 기능** (필수, 없으면 사이트로 안 보임)
2. **AI 약점 1줄 진단**
3. **메타 변화 알림**
4. **챔피언 마스터 챌린지** (게임화)

### Phase 3 — 신규 시장 (3~6개월)
1. **AI 실시간 음성 코칭**
2. **소셜 매칭**
3. **리플레이 AI 분석**

### Phase 4 — 수익화 본격화 (6개월+)
1. **프리미엄 모델** (광고 제거 + 심화 분석)
2. **인게임 오버레이 앱**
3. **개인 코치 매칭 플랫폼**

## 결론

한국 롤 시장은 **OP.GG 독점 + DAK/DeepLoL 틈새 공략** 구도. 우리가 진입한다면:

**❌ 하지 말 것**
- OP.GG가 잘하는 전적 검색 정면 승부 (이길 수 없음)
- 광고 떡칠 (유저가 가장 싫어함)
- 영어 우선 (한국 시장 포기)

**✅ 할 것**
- **밴픽 의사결정 도구**로 포지셔닝 (시장에 비어있음)
- **모바일 + 한국어 + 광고 최소화** (기본 차별점)
- **실용적 AI** (분석을 위한 분석이 아닌 게임 승리에 직접 도움)
- **틈새부터 잡고 확장** (Porofessor가 인게임 분석으로 살아남은 것처럼)
