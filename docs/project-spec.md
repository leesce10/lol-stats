# LOL Stats 기획서 (현재 구현 기준)

## 시스템 구조

```
사용자 브라우저
    ↓
Vercel (프론트엔드 + API 서버)
    ↓                ↓
Supabase DB     Riot Games API
(매치 데이터 저장)  (데이터 수집 원본)
```

---

## 1. 챔피언 통계 페이지 (`/`)

**목적**: 모든 챔피언의 랭크 성과를 한눈에 비교

**동작 방식**:
1. 페이지 접속 시 → Supabase DB에서 수집된 매치 데이터를 가져옴
2. 118경기(현재) 속 챔피언별 승패를 집계해서 통계 계산
3. DB에 데이터가 없으면 → 시뮬레이션 데이터를 폴백으로 표시

**사용자 기능**:

| 기능 | 설명 |
|------|------|
| 포지션 필터 | 전체/탑/정글/미드/원딜/서폿 선택 |
| 정렬 | 각 컬럼 클릭으로 오름차순/내림차순 |
| 검색 | 한글/영어로 챔피언 이름 검색 |
| 데이터 표시 | 승률, 픽률, 밴률, 게임수, 티어(1~5), 선픽점수(0~10) |
| 실데이터 표시 | 초록색 배지로 "실제 데이터 · 챌린저 N경기 기준" 표시 |

**데이터 출처**: 한국 서버 챌린저 솔로랭크 매치

---

## 2. 맞라인 분석 페이지 (`/matchup`)

**목적**: 내 챔피언 vs 상대 챔피언 매치업 분석 및 플레이 가이드

**동작 방식**:
1. 포지션 선택 → 해당 포지션 챔피언만 필터링
2. 내 챔피언 + 상대 챔피언 선택
3. 알고리즘이 매치업 계산:
   - 챔피언 클래스 상성 (탱커 > 어쌔신, 어쌔신 > 메이지 등)
   - 근거리/원거리 이점
   - 각 챔피언의 기본 승률
   - 이걸 종합해서 매치업 승률 산출

**사용자에게 보여주는 것**:

| 항목 | 예시 |
|------|------|
| 매치업 승률 바 | 제드 41.7% vs 야스오 58.3% |
| 유불리 판정 | "야스오 유리한 매치업" |
| 초반 분석 (Lv 1~6) | "균형" - 정글 개입이 중요합니다 |
| 중반 분석 (Lv 7~13) | "유리" - 오브젝트 싸움에 참여하세요 |
| 후반 분석 (Lv 14+) | "불리" - 중반까지 우위를 점하세요 |
| 핵심 포인트 | 3~4개 핵심 요약 |
| 플레이 가이드 | 5개 상세 팁 |

**현재 한계**: 매치업 데이터는 알고리즘 기반 (실제 매치업별 통계가 아님)

---

## 3. 조합 분석기 페이지 (`/team`)

**목적**: 게임 시작 전 양팀 5v5 조합을 입력하면 승률 예측

**동작 방식**:
1. 블루팀(아군) 5명, 레드팀(상대) 5명 챔피언 선택
2. 각 챔피언의 속성을 분석:
   - 이니시에이팅 / 포킹 / 한타 / 스플릿 / 보호 / 폭딜 / 회복 / 탱킹 / DPS / CC
3. 팀별 강점/약점 도출
4. 개인 승률 + 조합 시너지 점수를 합산해서 승률 예측

**사용자에게 보여주는 것**:

| 항목 | 예시 |
|------|------|
| 승률 예측 게이지 | 블루 56.3% vs 레드 43.7% |
| 블루팀 강점 | 강력한 이니시에이팅, 뛰어난 한타 능력 |
| 블루팀 약점 | 물리 데미지 편중 |
| 레드팀 강점 | 우수한 포킹 능력 |
| 레드팀 약점 | 프론트라인 부족 |
| 종합 분석 | 텍스트로 전략 요약 |
| 팀 미리보기 | 양팀 챔피언 초상화 |

## 4. 데이터 수집 시스템 (백엔드)

**API 엔드포인트**:

| URL | 역할 |
|-----|------|
| `GET /api/collect` | Riot API에서 챌린저 매치 수집 → Supabase DB 저장 |
| `GET /api/stats` | DB에서 매치 데이터 읽기 → 챔피언 통계 계산 → 반환 |

**수집 프로세스** (`/api/collect` 1회 호출 시):
1. 한국 서버 챌린저 리그 상위 15명 조회
2. 각 플레이어의 최근 솔로랭크 매치 ID 10개 조회
3. 새로운 매치의 상세 정보 수집 (플레이어당 최대 3개)
4. Supabase DB에 저장
5. 약 30~45개 새로운 매치 추가
6. API 호출 85회 이내로 제한 (Rate Limit 보호)

**현재 수집된 데이터**: 118매치, 111개 챔피언 통계
