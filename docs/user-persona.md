# 타겟 유저 페르소나

## 페르소나 1: 솔랭 트라이하더 (핵심 유저)
- **프로필**: 골드~다이아 구간, 솔랭 위주, 하루 3~5판
- **니즈**: "이번 판 내가 뭘 픽해야 이기나?"
- **사용 시점**: 챔피언 셀렉트 중 (시간 압박)
- **원하는 것**: 
  - 상대 매치업 승률 빠르게 확인
  - 조합 유불리 한눈에 파악
  - 현 메타 티어 리스트
- **불만**: 느린 로딩, 광고가 화면 가림, 정보 과잉

## 페르소나 2: 입문/복귀 유저
- **프로필**: 언랭~실버, 최근 복귀 또는 신규
- **니즈**: "요즘 뭐가 강해? 뭘 배우면 돼?"
- **사용 시점**: 게임 시작 전 공부할 때
- **원하는 것**:
  - 쉬운 챔피언 추천
  - 빌드/룬 가이드
  - 라인전 기본 가이드
- **불만**: 너무 복잡한 UI, 전문용어, 데이터만 나열

## 페르소나 3: 데이터 덕후
- **프로필**: 다이아+, 통계에 관심 많음
- **니즈**: "이 패치에서 메타가 어떻게 바뀌었지?"
- **사용 시점**: 패치 후, 분석할 때
- **원하는 것**:
  - 패치별 승률 변화 추이
  - 세부 통계 (상대별 승률, 아이템별 승률)
  - 높은 샘플 수 기반 신뢰할 수 있는 데이터
- **불만**: 샘플 수 부족, 데이터 업데이트 느림

---

## 우선순위
1순위: **솔랭 트라이하더** — 가장 큰 유저풀, 반복 방문 높음
2순위: **입문/복귀 유저** — 성장하면 1순위가 됨, SEO 유입 높음
3순위: **데이터 덕후** — 충성도 높음, 커뮤니티 파급력

## 핵심 UX 원칙
- **3초 안에 원하는 정보** — 밴픽 중에 쓰는 서비스
- **모바일 퍼스트** — 게임 밖에서 폰으로 봄
- **데이터는 많되, 표시는 단순하게** — 과잉 정보 지양
- **로딩 빠르게** — 느리면 바로 이탈
